## JSX = HTML with superpowers

```jsx
<Layout>
  <h1>This is my header</h1>
  <img src="/images/puppy.jpg" alt="Puppy image" />
  <button onClick={(evt) => console.log(evt)}>Click me!</button>
  <Avatar />
</Layout>
```

1. Mostly just HTML!
2. Components (`Layout` and `Avatar`) are capital case
3. Components may contain _children_
4. Event handling built-in without need to use JavaScript DOM selectors
